# 🛡️ honeypot attack detection
Python-based honeypot system for detecting and blocking brute-force login attacks using socket programming and real-time analysis


## 📌Overview
This project simulates a honeypot server that detects brute-force login attempts.

## Features
- Fake login system
- IP-based attack detection
- Username-based detection
- Global traffic monitoring
- Automatic blocking with cooldown

## How to Run

### Run server
python3 server.py

### Run client
python3 client.py
